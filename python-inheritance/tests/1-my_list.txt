=============================
Testing MyList class
=============================

Bu fayl `MyList` sinfini test etmək üçündür.

Gəlin sinfi import edək və yaradaq:
    >>> MyList = __import__('1-my_list').MyList
    >>> my_list = MyList()

Siyahıya elementlər əlavə edək:
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)
    >>> print(my_list)
    [1, 4, 2, 3, 5]

`print_sorted` funksiyasının düzgün işlədiyini yoxlayaq:
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]

Orijinal siyahının dəyişmədiyinə əmin olaq:
    >>> print(my_list)
    [1, 4, 2, 3, 5]
